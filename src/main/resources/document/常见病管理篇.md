# 常见病管理篇
#### 确诊高血压后，除了按时服药，日常生活中还需特别注意什么？
除遵医嘱服药外，生活方式干预至关重要。严格限盐，每日摄入量低于5克。坚持DASH饮食（多蔬菜、水果、低脂奶）。每周进行5-7次、每次30分钟的有氧运动。学会管理情绪和压力，避免情绪剧烈波动。每日定时自测血压并记录，复诊时提供给医生参考。例如，陈爷爷在服药的同时，坚持用限盐勺、每日打太极拳，他的血压控制得更加平稳，用药量也得以减少。
推荐课程：《高血压居家管理全攻略：从用药到生活》，课程深入讲解血压监测要点、饮食运动处方及药物依从性管理，帮助您实现血压长期达标。

#### 家中有糖尿病患者，如何科学安排一日三餐的饮食？
安排糖尿病饮食需掌握“定时定量、营养均衡”原则。确保每餐包含充足的蔬菜（尤其非淀粉类）、适量优质蛋白（鱼、豆、蛋、瘦肉）和粗细搭配的主食。优先选择低升糖指数（GI）的食物，如糙米、燕麦。烹饪方式以蒸、煮、炖、拌为主，避免油炸和糖醋。可使用“餐盘法”：一半是蔬菜，四分之一是主食，四分之一是蛋白质。张阿姨为患糖尿病的老伴准备餐食时，会用小碗精确分装主食，并保证餐餐有绿叶蔬菜，使老伴的餐后血糖波动明显减小。

推荐课程：《糖尿病营养厨房：手把手教您做控糖餐》，课程由临床营养师设计，提供食谱范例、食物交换份法与外出就餐技巧，让控糖饮食既健康又美味

#### 慢性胃炎患者，在饮食和保养上有哪些“红灯”食物和“绿灯”习惯？
“红灯”食物需警惕：避免辛辣、过酸、过甜、油炸食品及浓茶、咖啡、酒精。少吃粗纤维多的粗粮和未发酵的面食。“绿灯”习惯应养成：规律进食，细嚼慢咽，每餐七分饱。食物以温和、软烂易消化为主，如小米粥、山药、蒸蛋。注意腹部保暖，保持情绪舒畅。刘先生患有慢性胃炎，他戒掉了早餐喝浓咖啡的习惯，改为吃小米粥和馒头，并坚持饭后散步，胃部不适发作频率大大降低。

推荐课程：《胃的“养护说明书”：慢性胃炎与消化问题调理》，课程详细解析各类胃病的养护重点，提供饮食清单与情绪调节法，教您科学养胃，远离不适。

#### 家中老人患有骨关节炎，如何通过非药物方法缓解疼痛、改善功能？
非药物疗法是骨关节炎管理的基石。进行低冲击性的规律运动，如游泳、骑自行车、太极拳，以增强关节周围肌肉力量。控制体重，减轻关节负荷。注意关节保暖，可使用护膝等支具。居家进行关节功能锻炼，如直腿抬高、非负重下的屈伸活动。物理治疗如热敷、超声波等也有助缓解症状。吴奶奶膝盖疼痛，在医生指导下坚持每天在泳池中行走和做直腿抬高练习，几个月后，她上下楼的疼痛感显著减轻。

推荐课程：《骨关节炎居家康复训练营：不动痛，动对才轻松》，课程由康复治疗师示范安全有效的关节锻炼动作，提供疼痛自我管理策略，帮助患者提升活动能力和生活质量。

#### 如何帮助患有慢性病的家人建立良好的用药依从性，避免漏服、错服？
可借助工具和建立习惯。使用分药盒，提前将一周药物分装好。设定手机闹钟或利用用药提醒APP。制作清晰的用药清单，列出药名、剂量、时间，贴在醒目处。与家人一起了解每种药的作用和重要性，增加其主动性。定期检查药箱，陪同复诊。周阿姨为健忘的老伴准备了电子药盒，到点会自动响铃并弹出药格，同时女儿每天会打电话提醒，极大地改善了老伴的用药规律性。
推荐课程：《慢性病用药管理必修课：让服药更安全、更简单》，课程讲解药物储存、不良反应识别、依从性提升技巧及与医护沟通要点，成为家庭用药的智慧管家。
