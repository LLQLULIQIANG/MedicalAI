# 预防保健篇
#### 确诊高血压后，除了按时服药，日常生活中还需特别注意什么？
预防“三高”需从饮食、运动和生活习惯多管齐下。饮食上遵循低盐、低脂、低糖原则，增加全谷物、蔬菜水果和优质蛋白摄入。保证每周至少150分钟的中等强度有氧运动，如快走、游泳。同时，戒烟限酒，保持规律作息与健康体重，并定期监测血压、血脂、血糖指标。例如，王先生通过将日常食用油换成植物油、坚持每日快步走、并成功减重5公斤后，其体检时的临界高血压和血脂指标均恢复了正常。
推荐课程：《“三高”预防与生活方式管理》，本课程系统讲解科学饮食搭配、个性化运动方案及压力调节技巧，助您从源头建立健康防线，远离慢性病。

#### 怎样为自己和家人制定一份科学的年度体检计划？
制定体检计划需结合年龄、性别、家族史和个人生活习惯。基础项目应包括血常规、肝肾功能、血脂血糖和腹部B超。30岁以上或有家族病史者，建议增加心脏彩超、颈动脉超声及肿瘤标志物筛查。女性需重视乳腺和妇科检查，男性则应关注前列腺健康。可以咨询专业医生，进行风险评估后确定个性化方案。李女士为全家制定了差异化体检计划，为父亲增加了心脑血管深度筛查，为自己安排了乳腺专项检查，做到了疾病早发现。
推荐课程：《家庭健康守门人：科学体检规划指南》，课程详细解读各体检项目的意义，指导您根据家庭成员情况定制筛查方案，高效管理全家健康。

#### 职场人群如何应对和缓解久坐带来的健康风险？
应对久坐风险，关键在于“动”起来。每坐45-60分钟，必须起身活动5分钟，做简单的伸展。在工作间隙融入“微运动”，如靠墙静蹲、提踵练习。优化办公环境，使用可调节站立式办公桌。下班后进行有氧运动和力量训练，以补偿日间活动不足。例如，程序员小张设置了定时起身的闹钟，并在工位旁准备了弹力带，利用休息时间进行抗阻训练，有效改善了腰背酸痛和颈部僵硬的问题。
推荐课程：《办公室健康改造计划：告别久坐伤害》，课程提供全天候的职场运动方案与工位 ergonomics 调整技巧，助您在忙碌工作中也能保持身体活力。

#### 如何辨别网络上的健康信息真伪，避免被误导？
辨别信息真伪，首先要核查来源，优先选择权威医疗机构、官方卫健委或知名医学期刊发布的内容。警惕标题夸张、声称“治愈所有”或推销特定产品的内容。可运用“交叉验证”法，用多个可靠信源核实同一说法。对于个人健康决策，最终应咨询执业医师，而非依赖网络信息。赵阿姨在看到一个“神奇保健品”宣传后，通过在国家药品监督管理局官网查询其批号，发现它只是普通食品，从而避免了经济损失和健康风险。
推荐课程：《医疗健康信息素养课：做聪明的健康决策者》，课程传授鉴别健康谣言、评估信息可信度的核心方法，让您在海量信息中快速找到科学依据。

#### 家庭成员突发意外（如烫伤、异物卡喉）时，正确的现场急救步骤是什么？
保持冷静是关键。对于烫伤，立即用流动的冷水冲洗伤处15-20分钟降温，再以洁净纱布覆盖，切勿涂抹牙膏、酱油等。对于异物卡喉，若患者能咳嗽，应鼓励其咳嗽；若无法呼吸（海姆立克征象），应立即从背后环抱患者，快速向内向上冲击其上腹部。同时，务必第一时间拨打急救电话。孙先生的孩子不慎被坚果噎住，他迅速运用所学的海姆立克急救法成功将异物排出，为送医赢得了宝贵时间。
推荐课程：《家庭急救黄金时间：常用急救技能实战训练》，课程通过视频演示与模拟演练，手把手教学心肺复苏、外伤处理等必备急救技能，让您成为家人的安全守护者。
